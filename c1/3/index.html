微机原理填空题

# 微机原理填空题

  • 计算机主板上总线的发展情况：微型计算机的标准总线从16位的EISA总线，发展到32位的 EISA（扩展标准总线） 总线和 PCI（外设部件互联标准总线） 总线。

  • 编程语言发展的三个阶段： 机器语言、汇编语言、高级语言。

  • 8086微处理器按功能分为 EU（指令执行部件）、BIU（总线接口部件），两个单元相互独立，各自操作、并行执行，实现指令预取（流水线）。

  • 指令执行的步骤： 取指令、分析指令、执行指令。

  • 主存与cache的地址映射方式有： 全相连方式、直接相连方式、组相连方式。

  • 并行数据和串行数据的转换是用 移位寄存器。

  • SP 和 BP 寄存器与 SS 寄存器联合使用以确定堆栈中存储单元地址。

  • 执行PUSH AX指令后，SP发生了什么变化 SP-2。

  • 执行POP AX指令后，SP发生了什么变化 SP+2。

  • 堆栈的特点 先进后出/后进先出。

  • 执行HLT（处理器“暂停”指令）需要 一个 总线周期。

  • 执行MOV AX,[1000H]需要 二个 总线周期。

  • 执行ADD AX,[2000H]需要 三个 总线周期。

  • 8086/8088复位时，CS的值为 FFFFH ，IP的值为 0000H 。

  • 中断类型有 256 种，分为 内中断 、外中断。

  • 中断向量表在内存的前1KB中的地址范围是 0000H ~ 03FFH。

  • 一个中断向量占 4 个字节，高字节放 CS 、低字节放 IP。

  • 在系统中断，中断类型号是在 中断响应信号（INTA非） 的作用下送完CPU。

  • 中断源是指 引起中断的设备或事件 。

  • CPU在中断服务程序中需要响应比当前中断源更高级别的中断，则应在中断服务程序的开头加上 STI【开中断（IF置1）】 中断服务子程序的最后一条指令是 iret（返回主程序）。

  • CPU的能直接访问 主存 和 cache ，但是不能访问 外存 和 IO设备 。

  • BHE =0、A0=1时，cpu访问 奇地址体 。

  • BHE =1A0=0时，cpu访问 偶地址体 。

  • PSW寄存器中共有 6 位条件状态位，有 3 位控制状态位。

  • 8086微型处理器在访问四个当前段时，堆栈段的偏移量由 SP/BP 提供。

  • 控制器是微处理器的重要组成部分，下列哪项不是控制器组成部分 A 。【2020年】

    A.消息队列、B.程序计数器PC、C.指令寄存器、D.时序电路

  • 88\86执行RESET（复位）后CS= FFFFH 、IP= 0000H。

  • 一个完整的中断过程一般经历以下五个步骤：中断请求、中断优先级判定、中断相应、中断处理、中断返回。

  • 一个存储单元除具有一个唯一的 物理地址 外，还具有多个 逻辑地址 。

  • NMI（不可屏蔽中断）引脚所对应的中断类型码是 02H 。

  • 中断控制器8259可管理 可屏蔽中断 。

  • 实质上，微处理器对外设访问就是对外设 接口 中的端口寄存器进行访问。

  • 在8088CPU中，为了减少CPU等待取指令所需要的时间，设置了 指令队列 。

  • ROM是 非易失性存储器 ，RAM是 易失性存储器 。

  • 中断向量是中断服务程序的 入口地址 ，每个中断向量占 4 字节。8086中对应256类中断的中断向量存放在 中断向量表 中，占据内存从 00000H 到 003FFH 区间。

  • 在有多重中断的系统中，通常解决中断优先级的方法有 软件查询 、 硬件查询 和中断优先级编码三种。

  • 主存储器的技术指标有 速度 、 容量 和主存带宽。

  • 微处理器应包含的最基本功能部件是 算数逻辑单元 、控制器部件 和 存储器单元（寄存器） 。

  • 8086/8088在满足 当前指令结束 和标志位IF=1的条件下可以响应一个外部INTR中断请求。

  • 在8088CPU中，为了减少CPU等待取指令所需的时间，设置了 指令队列 。

  • 在有多重中断的系统中，通常解决中断优先级的方法有 软件查询 、 硬件查询 和 中断优先级编码 三种。

  • 保护断点就是保存 当前CS、IP的值 。

  • 8086CPU的PUSH指令执行时 先将SP-2，再压数据 。

  • 在8086最小工作方式下，典型的 8282芯片 功能是 地址锁存 。

  • 8288芯片的作用是 总线控制器 。

  • 8086根据所构成系统的大小不同，可以工作在最大方式或最小方式。在最大方式下，系统需要使用 总线控制器（8288） 来形成总线周期。

  • DRAM与SRAM比较 DRAM比SRAM价格低 。

  • 表示数据的存储器单元符号的地址为 变量 。

  • 在8086系统中，为了能正确返回被中断的程序，则当CPU响应外部中断请求并转向中断处理子程序前，应将 断点的地址 压入堆栈。

  • 在串操作指令前使用重复前缀指令REPE，终止串的重复操作的条件是 CX=0或ZF=0 。

  • 在8086系统中对单片8259A进行初始化时，必须设置的初始化命令字为 ICW1、ICW2、ICW4 。

  • 串行同步传送时，每一帧数据都是由 同步字符 开头的。

  • 实现微处理器与8259之间信息交换的是 数据总线缓冲器 与读写控制电路。

  • 调用程序和子程序之间的信息传送称为 参数传递 。

  • RS-232C是应用于 串行 二进制的数据通信设备和数据终端设备之间的总线接口。

  • 指令REPNE/REPNZ的功能是相等或为零时重复操作，退出的条件是 CX=0或ZF=1 。

  • n+1位有符号数x的补码表示范围为 -2^n^ <x<= 2^n^ -1

  • 世界上第一台电子计算机 ENIAC 机。

  • 8086微处理器的引脚信号中，不受最小/最大工作方式影响的控制信号是 RD 。

  • 8位有符号二进制补码表示范围是 -128—127 。

  • 8086CPU的地址总线有20位，可直接寻址1MB的存储空间，在独立编址方式下，可寻址 64KB 个I/O端口。

  • 属于串行通信接口芯片标准的是 USB 。

  • 使用INT 21H的2号功能显示1十六进制数码，需要 将该数码加30H或加37H在再送DL 。

  • 中断屏蔽寄存器的作用是 禁止8259相应某级中断申请传向CPU 。

  • 8255A工作于方式0，微处理器可以采用 无条件传送 和 查询传送 方式 。

  • 8259A可以接受CPU送来的 初始化（预置） 命令字和 操作 命令字 。

  • 在8086最小模式的典型配置中，需 2 片双向总线收发器8286，需 3 片地址锁存器8282。

  • 8253具有 3 个独立的 16 位的计数器。

  • 8位二进制反码10000000B 表示的十进制数为 -127 8位二进制补码10000000B表示十进制数为 -128 。

  • 全译码和线选法 存在地址重叠 问题，部分译码 不存在地址重叠问题 。

  • 74LS244是 常用的地址缓冲器 。

  • 2732EPROM是 4K*8 的芯片。

  • 下列有关程序控制传送方式的说法错误的是 A 。

  • A.中断传送方式不能实现多个外设并行工作。
  • B.查询传送方式又称“异步传送方式” 。
  • C.无条件传送方式又称“同步传送方式”。
  • D.中断方式数据传送完成后，程序会回到断点处继续执行。

  • 设模为2^8^,则53的补码为 35H ，-14的补码为 0F2H，-0的补码为 00H 。

  • 在段定义中，默认的定位类型是 PARA 。

  • 在进入DMA工作方式之前，DMA控制器被当作CPU总线上的一个 I/O接口 。

  • 中断向量号1AH在中断向量类型表中的逻辑地址是 0000H:0068H 。

  • 8255A的工作方式有基本型输入输出方式、字节选通I/O 和方式 双向I/O 方式。

  • 地址锁存器8282的选通信号STB应接CPU的 ALE 信号，输出允许端OE 应接 地 ;数据收发器8286的方向控制端T应接 DT/R ，输出允许端OE 应接 DEN 。

  • 当8253工作在 方式5 时，需要由外部脉冲触发开始计数。

  • 有关储存器映像I/O方式正确的是 C 。

  • A.称为I/O独立编址 B. I/O端口地址空间独立于存储器地址空间 C. 增加地址译码的复杂性 D. I/O指令类型较少
