报名流程

# 报名流程

3月11日上午9点——3月15日下午5点

# 1、考生注册

  • 打开河北省专接本报名系统网址：河北省专接本报名系统 (opens new window)

  • http://zjbks.hee.gov.cn/

  • 点击考生注册--->注册完成后开始登录

  • 用户名：本人身份证号码;密码：（初始密码）本人省份证号后六位

应及时登录系统修改密码

# 2、报名

重要的事说三遍：（填写时确认信息正确）

缴费确认后信息将不能修改！

缴费确认后信息将不能修改！

缴费确认后信息将不能修改！

登录完成后选择考生专业志愿填报

  • 点击 第一步：网上报名须知--->阅读完成后点击 我已阅知 进入第二步；

  • 点击 第二部：报考信息录入--->填写相应信息（填写时一定要认真核对填写信息）--->保存

  • 点击 第三步：报考信息确认--->查看已经填写的信息（确认无误后！）--->进入第四步

  • 点击 第四步：缴费状态查看--->勾选同意诚信协议确认（下载）--->填写缴费手机号--->点击入缴费系统进行缴费即可！

# 3、缴费（收费标准 ）

1、报名缴费成功后，不可退款

2、具体收费标准为：艺术、体育类每生120元、其他类别每生100元。

# 4、下一步

需要等待24小时，出现支付成功页面，自己 下载保存好诚信协议和考生报名表即可。
