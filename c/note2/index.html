# 第二章 控制语句

  • if…else… 条件语句
  • for()… 循环语句
  • whille()… 循环语句
  • do…whille() 循环语句
  • continue 结束本次循环
  • break 终止执行switch或循环语句
  • switch 多分支语句
  • return 从函数返回语句
  • goto 转向语句，在结构化程序中基本不用goto语句
